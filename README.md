2. Installation & Setup
Clone the Repository: Open your terminal and run:

Bash
git clone https://github.com/ryandavecaagoy-dev/hotel-website.git
```.
Move to XAMPP: Copy the entire hotel-website folder into your XAMPP installation directory: C:\xampp\htdocs\.

Start Apache: Open the XAMPP Control Panel and click Start next to the Apache module.

Access the Site: Open your browser and type: http://localhost/hotel-website/menu.php.
